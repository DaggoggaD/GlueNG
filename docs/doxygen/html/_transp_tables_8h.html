U64 get_zobrist_code ( Board * board)

Calculates the Zobrist code for a given board position.

Note
This function is not actually used in the search, as it is inherently slow. The actual hash key is updated incrementally in make_move/unmake_move.
Parameters
boardPointer to the current board.
Returns
The complete Zobrist hash key for the position.
