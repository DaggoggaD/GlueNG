\doxysubsection{Detailed Description}
This file contains the function declarations for calculating pseudo legal moves in the chess engine. 


\begin{DoxyItemize}
\item It includes functions to generate pseudo-\/legal moves for each piece type, as well as a function to check if a square is attacked by any pieces of a given color. The moves are represented in a compact format using bit packing, and the file also defines macros to extract move information from the packed move representation. 
\end{DoxyItemize}
