int beta,
int extension,
int ply,
int * outScore )

If the position is not in the transp table, we apply a null move (skipping our turn).

  • If the position is still in our favor (score >= beta), we can prune this branch.
  • Parameters
    boardPointer to the current board.
    entryPointer to the Transposition Table entry.
    depthThe remaining search depth.
    betaUpper bound of the search window.
    extensionCurrent depth extensions accumulated.
    plyThe current ply from the root.
    outScorePointer to store the result of the null move search.
    Returns
    True if a null move cutoff occurred, false otherwise.

◆ pst_lazy_evaluation()

int pst_lazy_evaluation ( Board * board,
int noPawnEval )

Evaluates current position based on piece position, using piece-square tables.

  • Also considers endgame phase for king evaluation. From white's perspective. Does not consider mobility scores, as they are heavier, so it's done only if the position is close to beta. Old, replaced with PeSTO evaluation.
  • Parameters
    boardPointer to the current board.
    noPawnEvalThe material evaluation without pawns (used to check endgame phase).
    Returns
    The static piece-square evaluation score.
